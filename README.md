# Random Decision App 

An iOS application built with **SwiftUI** and **Core Data** designed to eliminate decision paralysis. This project is developed as part of the **CPSC 411** requirements.

##  Project Overview
The Random Decision App allows users to input multiple choices (e.g., lunch options, movie titles, or tasks) and uses a randomization algorithm to select one.

##  Key Features
- **Summary View:** A dynamic list where users can add, view, and delete potential options.
- **Randomizer Engine:** A clean logic layer that ensures unbiased selection.
- **Data Persistence:** Integrated with **Core Data** to save user lists between sessions (Implementation in progress).
- **Drill-down UI:** Detailed view for individual options using `NavigationLink`.

##  Technical Architecture
- **Framework:** SwiftUI
- **Pattern:** Model-View-ViewModel (MVVM)
- **Database:** Core Data
- **Version Control:** Git/GitHub

##  How to Run
1. Clone this repository.
2. Open `RandomDecisionApp.xcodeproj` in **Xcode 15+**.
3. Select an **iOS Simulator** (e.g., iPhone 17).
4. Press `Cmd + R` to build and run.
